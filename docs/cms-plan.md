# CMS and Admin Plan

## CMS

Use Sanity CMS.

Editable content:
- Homepage
- About page
- Claudyo's Legacy
- Foundation page
- Gallery
- Media
- Blog
- Donation link
- Contact details
- Events
- Acts of Kindness

## Admin Dashboard

Protected by Google OAuth.

Routes:
- /admin/login
- /admin
- /admin/pages
- /admin/blog
- /admin/gallery
- /admin/media
- /admin/feedback
- /admin/settings

## Admin Features

Dashboard:
- Visitor stats
- Donation clicks
- Blog count
- Feedback count
- Gallery count

Pages:
- Edit page content
- Open Sanity Studio

Feedback:
- Approve / reject messages
- Mark urgent support messages

Gallery:
- Add images
- Categorise images
- Edit captions

Settings:
- Donation link
- Social media links
- Contact email
- Charity info
