# CB's World Foundation v2 Fresh Rebuild

Clean Next.js codebase using standard CSS, no Tailwind dependency, and the uploaded media organised into `public/`.

## Run locally

```bash
npm install
npm run build
npm run dev
```

## Pages

- Home
- Claudyo's Legacy
- Foundation
- Acts of Kindness
- Gallery
- Media
- Donate
- Contact
- Blog / Shop / Feedback placeholders
