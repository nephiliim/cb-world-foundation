## Project Vision

Build the official CB's World Foundation digital platform — not just a website.

It should be:
- Emotional and respectful
- Modern and professional
- Easy to edit like WordPress
- Built around Claudyo Jauad Lafayette's legacy
- Focused on ending knife crime, supporting bereaved families, education, awareness, community support, and acts of kindness

## Chosen Design

Use **Design B**:
- White, blue, black

- Storytelling-led
- Premium but warm

## Core Message

CB's World Foundation Charity exists to honour Claudyo Jauad Lafayette by:
- Tackling knife crime
- Supporting grieving families
- Educating young people
- Challenging stereotypes
- Providing grief and bereavement support
- Installing / promoting bleed control kits
- Building safer communities
