# Branding Guide

## Style Direction

Chosen direction: Design B.

The website should feel:
- Soft
- Hopeful
- Clean
- Trustworthy
- Emotional
- Professional
- Family-led

## Colours

Primary:
- Deep blue
- White
- Black

Accent:
- Light blue
- Gold highlights used sparingly

Suggested CSS variables:

--blue-dark: #0B2D5B;
--blue-main: #1155CC;
--blue-soft: #EAF3FF;
--black: #050505;
--white: #FFFFFF;
--gold: #D4AF37;
--text: #172033;
--muted: #667085;

## Typography

Suggested:
- Headings: Playfair Display, Georgia, or a clean serif
- Body: Inter, Arial, or system sans-serif

## Design Elements

Use:
- Large photography
- Rounded cards
- Soft shadows
- Blue section dividers
- Subtle animations
- Clear donation buttons
- Calm spacing

Avoid:
- Overly dark aggressive design
- Too much red
- Busy layouts
- Too much text on homepage
