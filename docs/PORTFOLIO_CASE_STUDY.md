# Portfolio Case Study

## Project
CB'S WORLD Foundation Charity Platform.

## Problem
The charity needed a professional website and a way to manage content without code changes.

## Solution
Built a full-stack charity platform using Next.js, React, TypeScript, Supabase, GitHub and Vercel.

## Features
- CMS
- Events
- Gallery
- News
- Legacy Wall
- Feedback
- Bleed Kit Map
- Drag and drop uploads
- Event automation
- Command Centre

## Skills demonstrated
Full-stack development, API design, database design, deployment debugging, version control, CMS design and product planning.
