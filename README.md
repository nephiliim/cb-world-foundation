# CB World Foundation V6 Complete Homepage

Copy these folders/files into the root of `C:\Projects\cb-world-foundation`:

- `app/page.tsx`
- `app/layout.tsx`
- `app/v6-homepage.css`
- `components/home/`

Then run:

```cmd
npm run build
git add .
git commit -m "Install complete V6 homepage"
git push
```

Important: do not copy these files into `public/`.
