# Media Library

The Media Library is the central asset store for CB'S WORLD Foundation Charity.

## Purpose

Upload once, reuse everywhere.

Media can be used for:

- Events
- News
- Gallery
- Homepage
- Legacy Wall
- Bleed Kit pages
- Future AI content generation

## Features

- Supabase Storage upload
- `cms_media` database table
- Search
- File type filtering
- Tags
- Categories
- Copy URL action
- Image/video/document support

## Future Improvements

- Choose from Media Library inside event/news forms
- Delete/archive media
- Used-in tracking
- AI search
- Auto-tagging
